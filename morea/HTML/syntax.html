Syntax

Before we can do any code in html there are some things we have to go over. One of these things is syntax.

What is syntax?

Syntax is a set of rules that must be followed. We are about to go over the syntax of html to which you will need for the rest of this module.

Elements/Tags

An HTML element is an individual component of an HTML document or web page. HTML document is comprised of multiple elements to provide content and structure to a document or webpage.

In HTML except for a few exceptions, all elements/tags must have an opening and closing declaration.

<tagname>Content goes here...</tagname>

An opening tag is declared when the tag name is encapsulated by a greater than and less than symbol.

<tagname>
<!-- An example of an opening tag-->

An closing tag is declared when there is a / before the tag name

</tagname>
<!-- An example of a closing element tag-->

The HTML element is everything from the opening tag to the end tag:

<p>My first paragraph.</p>
<!-- An example of a paragraph tag in HTML-->

Attributes

Attributes in HTML are instructions or characteristics of HTML elements that we can set and modify.

All attributes are made up of two parts:

  • Attribute name
  • Attribute value

We start with the attribute name followed by the equal(=) sign and in quotation marks(“”) the Attribute value.

<div attributeName="value"></div>

Every element in HTML has a set of attributes that can be modified or set. For Example: The <table> element has a attribute called sortable which sorts the data by each value.

In HTML there are attributes some attributes that are commonly found and used in many or almost all elements.

class

The class attribute is used to associate an element with a style sheet, and specifies the class of element. You will learn more about the use of the class attribute when you will learn Cascading Style Sheet (CSS). Many elements can have the same class value when you need to apply multiple styles to a similar element.

<div class="title">Junior Devleague</div>
<!-- -->

id

The id attribute is used to uniquely identify an element on a page. The id attribute is a little more strict with rules than its class counterpart.

  • The ID must start with a letter (a-z or A-Z)
  • All subsequent characters can be letters, numbers (0-9), hyphens (-), underscores (_), colons (:), and periods (.).
  • Each ID must be unique within the document.

When working with id’s in html, you cannot have multiple elements with the same value for id. I.E:

<div id="name"></div>
<div id="name"></div>
<!-- XXX: Cannot have multiple id's with the value "name" -->

title

The title attribute is used to provide extra information about an element. The information is most often shown as a tooltip text when the mouse moves over the element.

<p>
  Coders from all over the world gather in <span title="San Francisco">SF</span> for the annual <span title="Facebook">FB</span> developer conference.
</p>

href

The href tag is commonly used with the <link> and <a> tags to provide a reference to the location of a external file, webpage, or anchor point in the webpage.

This attribute has 2 types of values

  • URL or path - Location of a external file or webpage
  • anchor point - Location on the current web page to be navigated to.
<a href="#idName">idName</a>
<a href="https://www.google.com">google.com</a>
